# Markdown_Cheatsheet
## Headers

Type this:

```
# Main Heading
## Sub Heading
### Sub Sub Heading
#### Sub Sub Sub Heading
##### Sub Sub Sub Sub Heading
###### Sub Sub Sub Sub Sub Heading
```

Get this:
# Main Heading
## Sub Heading
### Sub Sub Heading
#### Sub Sub Sub Heading
##### Sub Sub Sub Sub Heading
###### Sub Sub Sub Sub Sub Heading

## Formatting 

Type this:

```
Emphasis, aka italics, with *asterisks* or _underscores_.

Strong emphasis, aka bold, with **asterisks** or __underscores__.

Combined emphasis with **asterisks and _underscores_**.

Strikethrough uses two tildes. ~~Scratch this.~~
```

Get this:

Emphasis, aka italics, with *asterisks* or _underscores_.

Strong emphasis, aka bold, with **asterisks** or __underscores__.

* Item 3

# Code

Type this:

```Inline `code` has `back-ticks around` it.```

Get this:

Inline `code` has `back-ticks around` it.

For blocks of code, type this:

    ```python
    s = "Python syntax highlighting"
    print(s)
    ```

Get this:

```python
s = "Python syntax highlighting"
print(s)
```

## Links
Type this:

```[The text I want you to click](http://www.google.com)```

Get this:

[The text I want you to click](http://www.google.com)

## Images
If you don't need to resize, type this:

```![GitHub Octocat](media/github-octocat_150x100.jpg)```

Get this:

![GitHub Octocat](media/github-octocat_150x100.jpg)

If you need to resize, type this:

```<img src="media/github-octocat_150x100.jpg" width="75">```

Get this:

<img src="media/github-octocat_150x100.jpg" width="75">
